Project Details

Musinsa 랭킹 데이터 분석

평소 관심이 많던 대한민국 최대 규모의 온라인 편집숍인 무신사 사이트의 상품 랭킹페이지를 크롤링, 데이터 분석을 함으로써 구매 시 어떤 브랜드가 가장 합리적인지를 판단하는 데 도움이 될 수 있을 지 그래프 및 표로 볼 수 있다면 좋겠다 생각하고 시작하게 된 데이터 분석 프로젝트입니다.

  • 프로젝트 방식: Python
  • 수행기간: 2023-1 ~ 2023-02
  • github: https://github.com/kildonghyeon/Musinsa_Analysis
프로젝트 소개
  • 주요 사항 - BeautifulSoup, requests 라이브러리를 통한 웹크롤링으로 분석할 데이터수집
    - 후처리한 데이터를 바탕으로 만들어낸 WordCloud
    - ggplot, matplotlib, seaborn을 활용한 데이터 시각화 작업

개발환경
PROGRAMMING LANGUAGE OPEN SOURCE ETC
Python 3.6.13 Jupyter Notebook, Anaconda BS4, Numpy, Pandas, Requests,
Plotnine, Matplotlib, Seaborn, WordCloud

작업 내용
웹 크롤링과 데이터 후처리 WordCloud
  • requests, BeautifulSoup 라이브러리를 사용해 무신사 URL 호출, HTML 파싱
  • 순위, 제품, 브랜드, 가격, 좋아요 수의 값을 가져옴
  • 카테고리 별 리스트 생성 후 하나의 데이터프레임으로 생성
  • 'Rank', 'Good', 'brand', 'price', 'like'라는 이름의 총 5개의 컬럼으로 이루어진
    데이터프레임이 되도록 후처리 작업
  • 데이터프레임 내 가장 많이 랭킹에 오른 브랜드 기준으로 제작
  • Mask: 무신사 모델 클리핑 마스크
  • Font: 배민한나체 Pro
  • Color: 랜덤한 Grey 색상
분석하고자 한 타이틀 시각화
  • 가장 많이 랭킹에 오른 브랜드
  • 위 타이틀의 상위권 Top20의 상품수 비교
  • 상품수 상위권 Top20의 최소, 평균, 최대 금액 비교
  • 평균 좋아요 수 상위권 Top20 브랜드의 가격비교
  • 상품수 상위권 Top20의 가격대 별 평균 좋아요 수 비교
  • ggplot을 사용한 막대 그래프와 다중 막대 그래프
  • matplotlib와 seaborn을 사용한 파이 차트
  • matplotlib을 사용한 막대 그래프
