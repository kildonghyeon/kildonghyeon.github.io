MODA: 편의점 행사 정보 정리 사이트

모다(MODA)는 모두, 모으다라는 뜻이 있는 단어로 여러 편의점들의 행사 정보들을 한 곳에 모아 이용자가 보다 더 편하게 정보를 열람할 수 있는 커뮤니티성 웹사이트를 목표로 만든 Spring Framework 팀프로젝트입니다.

  • 프로젝트 방식: Spring MVC
  • 수행기간: 2022-11 ~ 2023-02
  • github: https://github.com/kildonghyeon/springConn
  • Website: http://localhost:8080/CVSWeb/
프로젝트 소개
  • 수행인원: 4명
  • 담당파트: 행사, 자유게시판 / 웹 크롤링 데이터 수집 전반 / 유지보수 및 보완
  • 개발 주요 사항 - Jsoup 라이브러리를 이용한 각 편의점 행사상품, 이벤트, 위치정보 등 웹크롤링
    - 웹크롤링을 통해 가져온 데이터 후처리 및 자동 업로드 기능 구현
    - Kakao API와 연동해서 모든 편의점 위치 검색

개발환경
SCRIPT DATABASE PROGRAMMING LANGUAGE FRAMEWORK IDE ETC
JSP, HTML, CSS(BootStrap),
JavaScript(jQuery) 		Oracle XE 11g Java 11 Spring Eclipse (STS 3.9.18) Ajax, MyBatis, EL/JSTL, Jsoup,
Apache Commons FileUpload,
JSON-simple, KakaoMap API, ...

담당구현내용
자유게시판 행사게시판
  • 글쓰기, 수정, 삭제 기능 구현
  • 글쓰기, 수정 시 Ajax를 사용하여 빈 칸 체크기능 구현
  • 게시글 클릭 시, 조회수 증가와 동시에 상세페이지로 이동
  • 관리자의 경우 게시글 작성 시, 공지글을 올릴 수 있도록 체크박스 추가
  • 글 검색기능 구현
  • 댓글, 대댓글 기능 구현
  • 부적절한 게시글, 댓글의 경우 회원들이 신고 가능하도록 신고버튼 구현
내용 상세보기
  • 전반적인 내용은 자유게시판과 동일
  • 관리자 계정일 경우만 글쓰기 기능을 사용할 수 있도록 코드작성
  • 사진첨부기능 구현
  • 카테고리와 검색어에 따른 검색기능 구현
  • 잘못된 정보가 올라올 시, 회원들이 오류신고가 가능하도록 신고문구 수정
내용 상세보기
웹크롤링 유지보수 및 보완
  • 행사상품 => 상품명, 상품가격, 행사유형, 상품이미지을 크롤링하여 받아옴
  • 이벤트=> 이벤트제목, 이벤트내용, 이벤트이미지, 이벤트기간을 크롤링하여 받아옴
  • 위치정보 => 지점명, 주소을 크롤링하여 받아옴
내용 상세보기
  • 로그인, 로그아웃 시, 무조건 메인페이지로 이동 => 로그인, 로그아웃을 한 페이지로 이동
  • 로그인 체크 방식 동기 처리 => 비동기 처리로 수정
  • 자동 행사 등록 시 중복되는 데이터 상관없이 추가 => 중복되지 않는 데이터만 추가되도록 수정
